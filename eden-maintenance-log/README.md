# The Adèle Archive: EDEN Maintenance Log

A static, front-end interactive archive interface for **The Adèle Archive**.

## Version

v3 language-access build.

## What changed in v3

- Added an opening language access page.
- Chinese and English archive profiles are selectable before boot.
- EDEN revision, raw residue, operator note, command feedback, final protocol, and final receipt now follow the selected language profile.
- English text is written as an archive-access version, not a line-by-line translation of the Chinese text.
- The v2 ending logic remains: seven maintenance choices form the main behavioral trace, while Final Protocol only adds weighted pressure.
- Final command conflict detection remains active.

## How to run

Open `index.html` directly in a browser, or deploy the folder to GitHub Pages.

No build step is required.
