Visitor Notice / 参观须知

TEMPLE-APPROVED TEXT

Official Guide / 官方导览

为维护母神殿的神圣秩序与公共参观安全，请所有访客遵守以下规定： 请在圣母安息厅与十二子苏醒室内保持静默。 请勿触摸展柜、封存容器、复原装置或任何标注为限制接触的设施。

未经授权，请勿在展区内录音、誊写祷词、复制展签或改写导览文本。 请勿向十二子影像提出私人问题。 请勿使用“强制”“实验”“约束”“不愿意”等未经认证的历史表述。

若您在旧实验区附近听见异常声响、发现未标注通道，或看到残留编号，请立即离开该区域并通知工作人员。

PUBLIC NOTICE // MT-VISIT-01

Archive Notes / 档案批注

现存早期版本的参观须知中，未发现“私人问题”“未经认证表述”与“异常声响”相关条款。 以上限制内容出现在母神殿开放后的第三次文本修订中。

内部访客系统记录显示，出口留言、停留时长、路径偏离与关键词均曾被纳入分析。 含有“阿黛勒”“不愿意”“谁决定的”等词汇的留言，会被系统标记为“不稳定反馈”。

“维护参观体验”在部分管理记录中对应：叙事校正、情绪安抚与低强度记忆干预。

Archive Notes / 档案批注

该术语表最初并非面向普通访客，而是用于讲解员、档案管理员与圣典誊写者培训。

术语替换改变了事件责任归属。“疼痛”被归入“奉献”，“约束”被归入“保护”，“拒绝”被归入“情绪波动”。

该表格不只规范语言，也规范了可被公开承认的事实范围。

05

Exhibition Sections / 主展区

01

圣母安息厅

Hall of Sacred Rest

TEMPLE-APPROVED TEXT

Official Guide / 官方导览

圣母安息厅是母神殿的核心纪念空间。据认证圣典记载，圣母阿黛勒在此完成最后恩典，并将自身的温度、呼吸与沉默交还给十二子，使其进入稳定苏醒阶段。

中央白石台为后期礼仪复原设施，用于标示圣母安息时的象征方位。请访客在本厅内保持静默，并依照地面指引完成环形参观。

母神殿提醒您：圣母并未离去。她以被纪念、被继承与被共同维护的形式继续存在。

EXHIBIT TEXT // HSR-01

Archive Notes / 档案批注

原始结构图显示，该区域最早标注为“生命维持约束区”。现有白石台并非原始设施，其下方仍保留接口槽、固定环痕迹、管线入口与体征同步端口。

早期记录未使用“安息”一词。对应表述包括：“生理活动终止”“系统接入失败”与“主体生命指标不可逆下降”。

“最后姿态复原”参考的是圣典插图，而非完整现场记录。

02

十二子苏醒室

Chamber of the Twelve

TEMPLE-APPROVED TEXT

Official Guide / 官方导览

十二子苏醒室记录了文明新晨的第一阶段。根据神殿认证叙事，十二名孩子在圣母完成安息后依次苏醒，并由此开启后圣母时代的公共生命秩序。

每一座苏醒圣龛对应一名孩子，每一道光环用于标示其苏醒顺序与纪念编号。为维护展区秩序，请勿呼唤未经认证的私人姓名。

本展区为静默参观区域。请勿向影像、投影或互动装置提出私人问题。

EXHIBIT TEXT // CTT-02

Archive Notes / 档案批注

早期技术日志将本区称为“孵化槽列阵”。“苏醒圣龛”一词为后期公共展陈替换用语。

原始设备记录显示，十二个槽位属于人工孵化与生命稳定系统。第十二槽曾出现非同步反应，系统标记包括：“异常”“不稳定”“拒绝同步”“需隔离观察”。

关于孩子们是否保留苏醒前记忆的研究，在神殿建立后被终止。

  • 十二座苏醒圣龛
  • 苏醒顺序光环图
  • 第十二槽残损编号
  • 首次呼吸记录

请勿询问孩子们是否曾同意成为证据。

03

火种封存区

Seedfire Containment Gallery

TEMPLE-APPROVED TEXT

Official Guide / 官方导览

火种封存区为母神殿安全等级最高的公开展区。根据神殿说明，火种是圣母留给文明的最后试炼，曾引发偏离、混乱与不可预测的意志波动。

中央展柜中陈列的是最后一管火种的复原模型。真实样本不对公众开放，并由圣典委员会、安全部与技术保存组共同监管。

为确保参观安全，请勿靠近封存柜，请勿将手掌贴于玻璃表面，请勿尝试与展柜进行接触性互动。

EXHIBIT TEXT // SFG-03

Archive Notes / 档案批注

“污染”一词最早出现于母神殿安全部报告。目前可见的阿黛勒笔记中，未发现她以该词描述 Seedfire。

残存笔记显示，她曾将 Seedfire 记录为一种“被保留下来的拒绝能力”。后续公开导览均删去了“拒绝”相关表述。

火种机制可能涉及意志、自我命名、非服从行为与叙事断裂。神殿将其定义为危险物，但该定义未必来自原始研究者本人。

  • 火种封存柜
  • 最后一管 Seedfire 复制模型
  • 安全部污染等级牌
  • 火种扩散路线图

如果服从是一种病，火种是什么？

04

圣典初稿展柜

First Gospel Draft Display

TEMPLE-APPROVED TEXT

Official Guide / 官方导览

本展柜保存了《母神圣典》的早期草稿与修订版本。这些文本记录了圣母之言如何经过誊写、整理、祷告与校订，最终成为文明共同语言。

每一道修订痕迹都体现了后世对恩典叙事的持续完善。神殿认为，修订并非对原文的背离，而是对圣母慈悲更准确、更庄严的理解。

请勿在本展区进行未经授权的文本比对、摘录或再解释。

EXHIBIT TEXT // FGD-04

Archive Notes / 档案批注

多处修订显示，阿黛勒的第一人称语句被改为第三人称神圣叙述。

“我不愿意”被改为“圣母愿承众苦”。“不要再记录我”被改为“圣母进入慈悲静默”。“这不是爱”被改为“圣母以凡人无法理解的方式继续爱着我们”。

“实验记录”被替换为“圣言初稿”。该替换改变了文本的来源属性，也改变了读者对责任主体的判断。

  • 《母神圣典》第一版草稿
  • 红色修订笔迹
  • 术语替换表原件
  • 被撕下的原始语句残片

她说话。他们修订。后人祈祷。

05

母神寝室复原区

Reconstructed Chamber of the Mother

TEMPLE-APPROVED TEXT

Official Guide / 官方导览

母神寝室复原区呈现圣母生前短暂停歇的生活空间。本区根据遗物清单、圣典旁注与后世纪念图像重建，用于帮助访客理解圣母在日常层面的温柔与承担。

展区内陈列有窄床、水杯、织物、便签与睡眠相关记录的复原件。这些物品共同构成圣母继续履行职责前的安宁片刻。

请勿触摸床沿、织物或桌面物品。请勿将本区称为未经认证的私人房间。

EXHIBIT TEXT // RCM-05

Archive Notes / 档案批注

原始清单显示，此处并非普通寝室，而是生活监测单元。饮水、睡眠、体温、梦话、药物摄入、书写行为与情绪波动均被记录。

房间中至少存在三处监控点。多份记录显示，阿黛勒长期处于睡眠不足、疼痛管理不足与持续观察状态。

桌面便签“remember to eat lunch”不应被解释为神学文本。它更可能是一条日常自我提醒。

  • 窄床复原装置
  • 未喝完的水杯
  • 旧毯子
  • “remember to eat lunch”便签
  • 睡眠监测表

她不是圣像。她只是很久没有睡好。

06

阿黛勒遗物展

Relics of Adèle

TEMPLE-APPROVED TEXT

Official Guide / 官方导览

阿黛勒遗物展陈列圣母生前使用、接触或被认为与其恩典相关的珍贵圣物。每一件展品都见证了她对文明的爱、对十二子的慈悲，以及对未来的无私承担。

神殿以永久保存的方式维护这些圣物，并通过编号、封存、修复与公共展示，使其从私人遗存转化为文明共同记忆。

请勿触摸展柜。请勿对展品进行未经授权的私人解释、重新命名或情绪化归属。

EXHIBIT TEXT // ROA-06

Archive Notes / 档案批注

早期遗物整理记录缺失。目前无法确认哪些物品被保存、哪些被移交、哪些被销毁，哪些在进入展柜前经过重新命名。

部分展品可能曾作为证据、生活用品、研究材料或儿童物品存在，而非“圣物”。

“止痛药盒”被解释为圣母承担痛苦的象征，也可能是疼痛管理失败的证据。“Noah 的儿童画”首先属于一个孩子，不属于神殿叙事。

  • 烧毁的实验笔记
  • Noah 的儿童画
  • 给 Messiah 的未寄出信
  • 止痛药盒
  • 旧毯子

禁止进入的旧实验区

Restricted Former Laboratory

TEMPLE-APPROVED TEXT

Official Guide / 官方导览

旧实验区因结构老化、系统不稳定与安全维护需求，暂不对公众开放。本区域不属于母神殿推荐参观路线，也不构成现行圣母安息叙事的必要组成部分。

为保障参观安全，请勿靠近封闭门、扫描锁、维修通道或地下楼梯。若您误入旧实验区周边，请保持冷静，并沿最近的公共路线返回主展区。

请勿阅读墙面残留标记。请勿记录未认证编号。请勿回应门后声音。

Archive Notes / 档案批注

旧实验区是母神殿最早的空间核心之一。建筑下层仍保留实验台、人工孵化设备、旧式培养皿、封存管线与生命维持系统接口。

早期平面图显示，现行圣所可能由实验室、孵化中心、生命维持区与档案控制室逐步改造而成。

“不构成现行圣母安息叙事的必要组成部分”并不等于“与阿黛勒无关”。

Exit Prayer Station / 出口祷告区

您即将离开母神殿。离场前，您可以在出口祷告区写下一句献给圣母的话。 您的留言将进入神殿公共祝祷数据库，并可能在未来展览中匿名展示。

“圣母阿黛勒，感谢您为我们……”

Archive Notes / 档案批注

早期出口系统收集访客留言，用于监测公众对母神叙事的接受程度。留言内容、提交时间、路径偏离情况与关键词均可被关联记录。

含有“阿黛勒”“不愿意”“约束”“谁决定的”“她不是神”“她是人”等词汇的留言曾被自动隐藏。部分留言未删除，而是转入“叙事风险样本库”。

“愿圣母记得您”并非原始反馈语。早期版本为：“您的信息已被记录。”
